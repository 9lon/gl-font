Basic gl-styles Demo

How to use.
  1. You want to include 'gl-styles' into tag style css you file.
  2. You write css font-family into tag you want.
วิธีใช้งาน
  1. คุณต้องแทรกสไตล์ 'gl-styles' ลงในแทรก style ของไฟล์ที่คุณต้องการ
  2. เรียกใช้ css font-family ลงในแท็กที่คุณต้องการใช้งาน
Exsample Code
div.ex { font-family: 'rsuregular', sans-serif;
-webkit-font-smoothing: antialiased; }
